CSS: 글꼴, 텍스트, 리스트, 테이블

1.글꼴

 글꼴 관련 속성
font-family → 글자체 종류
font-size → 글자 크기
font-style → 글자 스타일
font-variant → 작은 크기의 대문자로 변환
font-weight → 글자 굵기
font → 글자 관련 속성(font-*)의 일괄 지정
@font-face → 사용자 정의 글꼴

 폰트 지정
폰트패밀리 → serif(명조 계열), sans-serif(고딕 계열), cursive(필기 모양의 글꼴), fantasy(장식이 된 글꼴), monospace(고정폭 글꼴)
일반폰트명 → arial, 궁서체, "Timies New Roman"...

☞ 예시
p{font-family:궁서체, "Times New Roman", serif}

 font-size 속성
절대크기: xx-small, x-small, small, medium, large, x-large, xx-large
상대크기: smaller, lager
절대크기 단위: cm, mm, in, pt, pc, px
상대크기 단위: em(부모요소의 글꼴 크기=1em), ex, %

 font-style, font-variant 속성
☞ font-style → 글자 모양 지정 / nomal, italic(기울어진 형태), oblique(기울어진 형태)
☞ font-variant → 작은 크기의 대문자로의 변환 여부

 font-weight 속성 → 글자의 굵기
normal, bold, bolder, lighter, 100, 200, ...800,900

 font 속성
☞ 글꼴 관련 속성(font-*)을 한꺼번에 지정하는 속성
font: [-weight값 | -style값 | -variant값] -size값 [/line-height값] -family값;

예시→ .style{font:700 italic small-caps 25pt /10pt sans-serif}

 @font-face 규칙
사용자 정의 글꼴을 사용하기 위한 규칙
☞ @font-face{font-family: "사죵자 지정 글꼴명"; src:url("글꼴의 경로와 파일명");}
@font-face{font-family:'myFont'; src:url(myFont.otf);}

 예시

Cascading Style Sheets: font

font관련 속성을 사용하려면 폰트(서체), 글자 크기, 스타일, 굵기, 작은 대문자로 지정 등과 같은 글자 자체의 모양을 지정할 수 있다

속성의 종류: font-family, -size, -style, -weight, -variant

2. 텍스트

 텍스트 관련 속성
letter-spacing: 문자 사이의 간격 → mormal, 길이(음수 가능)
word-spacing: 단어 사이의 간격 → normal, 길이(음수 가능)
line-height: 문장의 줄 간격 지정 → normal, 길이, 숫자(현재 글꼴 크기의 상수배), %(현재 글꼴 크기의 백분율)
text-align: 텍스트의 수평 정렬 방식 지정 → left, right, center, justify(양쪽 정렬)
text-align-last: 텍스트 마지막 줄의 정렬 방식 지정 → auto(=left), right, center, justify, start(라인의 시작 부분에 정렬), end(라인의 끝 부분에 정렬) → 이 속성을 사용하려면 먼저 text-align을 "justify"로 지정해야 동작한다
text-indent: 텍스트 블록에서 첫 번째 줄의 들여쓰기 지정 → 길이(기본값 0), %(부모 요소의 폭에 대한 백분율)
text-transform: 텍스트의 영어 알파벳 표기 방식 지정 → none, capitalize, upptercase, lowercase
text-decoration-line: 텍스트 장식과 관련된 선의 종류 지정 → none, underline, overline, line-through → 하나 이상의 속성값을 동시에 사용 가능
text-decoration-color: 선의 색상 지정
text-decoration-style: 선의 스타일 지정 → solid(실선), double(이중선), dotted(점선), dashed, wavy(물결)
text-decoration-thickness: 선의 두께 지정 → auto, 길이, %
text-decoration-*의 일괄 지정

텍스트 장식 line-through red wavy 3px

일괄 지정 underline overline blue dotted 20%

text-shadow: 텍스트에 그림자 효과 지정 → none, 수평위치(h>0 오른쪽), 수직위치(v>0 아래쪽), 흐림정도(그림자가 번지는 정도), 색상
text-overflow: 요소 박스 영역을 벗어난 텍스트의 표시 방식 지정 → clip(표시하지 않음), ellipsis(...)

지금까지 설명한 SELECT문으로는 레코드 단위의 데이터를 검색한다

word-wrap: 단어가 길어서 요소 폭을 넘을 때 단어를 분리해서 줄바꿈을 수행할지 여부 지정 → normal(미수행), break-word
vertical-align: 요소의 수직 정렬 방식 지정 → 길이, %, baseline, sub, super, top, middle, botton, text-top, text-bottom

3. 리스트

속성설명
list-style-type 항목 마커의 종류 지정
(ul 요소)값 → none, disc, circle, square (ol 요소)값 → decimal, lower-alpha, upper-alpha, lower-roman, upper-roman 등
list-style-position 항목 마커의 위치 지정
값 → inside, outside
list-style-image 이미지를 항목 마커로 지정
값 → none, url(이미지파일)
list-style 리스트 관련 속성(list-style-*)이 일괄 지정
  • 알고리즘

4. 테이블

 border속성
table, td, th 요소에 대한 테두리 지정

 table-layout속성
셀 안의 내용의 크기에 따른 셀의 크기 변화 여부 지정
auto → 셀의 폭이 셀의 내용의 크기에 따라 자동으로 결정
fixed → 셀의 내용의 크기와 상관없이 테이블/셀의 폭에 의해서만 결정

 border-collapse속성
테이블 테두리와 셀 테두리를 하나로 합칠지 여부 지정 separate → 두 테두리를 분리해서 표시
collapse → 두 테두리를 하나로 합쳐서 표시(border-spacing속성과 empty-cells속성은 무시됨)

 border-spacing속성
인접한 셀 테두리 사이의 간격 지정

 caption-side속성
테이블 캡션의 위치 지정 → top, bottom

 empty-cells속성
빈 셀의 테두리와 배경의 표시 여부 지정 → show, hide
