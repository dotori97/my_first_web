CSS: 박스모델, 테두리

1.박스 모델

마진 → margin(-top, -right, -bottom, -left)
패딩 → padding(-top, -right, -bottom, -left)
요소의 크기 → width, height, max-width, min-width, max-height, min-height
요소의 위치 → position, top, bottom, left, right, z-index
화면 표시 → display, overflow, visibility
플로팅 → float, clear
박스의 크기 → box-sizing, resize

 박스모델이란?
모든 HTML요소(div h4 p 등)를 사각형 형태의 박스로 취급
css를 통해 각 박스의 위치, 크기, 색상 등을 지정
웹 페이지의 레이아웃을 구성하는 중요한 개념
 마진 속성
박스의 외부 여백 지정
모든 방향(상우하좌)에 대해서 일괄 지정
margin:길이(기본값 0), %, auto
margin-방향

☞ 속성값의 개수에 따른 마진의 적용 방향
4 → 상 우 하 좌
3 → 상 좌우 하
2 → 상하 좌우
1 → 상하좌우

모든 방향(상우하좌)에 대해서 일괄 지정
padding:길이(기본값 0), %
padding-방향: 길이, %

 요소의 크기: width, height 속성
요소의 콘텐츠가 표시되는 영역의 폭과 높이 지정
값 → 길이, %, auto
두 가지 속성 중 하나만 지정되면 나머지 하나는 자동으로 크기 결정
 요소의 위치: position 속성
요소 배치의 기준이 되는 위치(요소의 위치 설정 방식) 지정
static → 페이지의 정상적인 흐름에 따라 현재의 위치가 요소의 위치가 됨(별도의 위치 지정/변경 불가)
absolute → 브라우저의 왼쪽 상단 모서리를 기준으로 지정한 위치만큼 이동하여 요소를 배치
relative → 해당 요소의 현재 위치를 (0.0)으로 정하고, 이를 기준으로 지정한 위치만큼 이동하여 요소를 배치
fixed → 브라우저 윈도우를 기준으로 지정한 위치만큼 이동하여 요소 배치(스크롤해도 움직이지 않음)

값 → auto, 길이, %(포함 블록에 대한 백분율)

 z-index 속성
요소 간의 겹쳐지는 순서 지정
auto, 정수(큰 값일수록 앞에 위치. 음수 가능)

 화면 표시: display 속성
요소를 어떻게 표시할 지를 지정 → 사용할 박스의 종류
none → 요소를 화면에 표시하지 않음
inline → 요소를 인라인 요소로 취급(줄바꿈 미수행)
block → 요소를 블록 요소로 취급(박스 앞뒤로 줄바꿈 수행)
inline-block → 인라인 수준의 블록으로 취급(박스 자체는 인라인 박스, 내부는 블록 형식)
list-item → 블록 박스를 목록의 항목과 같이 표시

첫 번째 span 요소 두 번째 span 요소 세 번째 span 요소 네 번째 span 요소

 화면 표시: overflow 속성
콘텐츠가 요소 박스를 벗어나는 경우의 처리 방법 지정
height 속성이 지정된 블록 요소에 대해서만 적용 가능
visible, hidden, clip, scroll, auto

☞ overflow-x, overflow-y 속성
→ 수평 및 수직 방향으로 구분해서 오버플로의 처리 방법 지정

 화면 표시: visibility 속성
요소 박스의 화면 표시 여부 지정
visible
hidden → 화면에 표시하지 않지만, 페이지에서 원래의 공간은 그대로 차지
collapse → hidden과 동일. 단, 테이블 요소의 경우에는 행이나 열 제거

 플로팅: float 속성
콘텐츠의 일반적인 흐름을 벗어나서 부모 요소 영역을 기준으로 해당 요소를 왼쪽/오른쪽에 배치하도록 지정
none, left, right

 플로팅: clear 속성
float 속성의 영향을 받은 요소의 흐름을 해제
none, left, right, both

float:left → clear:left or clear:both
float:right → clear: right or claer:both

float 속성은 콘텐츠의 일반적인 흐름을 벗어나서..

clear 속성은 float속성의 영향을 받은 요소의 흐름을 해제..

 박스의 크기: box-sizing 속성
width/height 속성을 사용하여 요소의 폭과 높이를 지정할 때 패딩과 테두리를 포함시킬지의 여부 지정
content-box → 오직 콘텐츠 영역만의 크기(패딩과 테두리 미포함)
border-box → 패딩과 테두리를 모두 포함

 박스의 크기:resize 속성
사용자가 요소 박스의 크기를 조절할 수 있도록 지정
none, both(폭/높이), horizontal(폭), vertical(높이)
"overflow: hidden/ scroll/ auto" 속성과 함꼐 사용해야 함

2. 테두리

 border 관련 속성
속성설명
border-width 테두리의 굵기 지정
medium, thin, thick, 길이
border-style 테두리의 스타일 지정
none, hidden, dotted, dashed, solid, double, groove, ridge, inset, outset
border-color 테두리의 색상 지정
색상(검정), transparent
border border-* 속성을 한꺼번에 지정
border-style속성값은 필수
방향별 속성: border-top-width:thin, border-right-width:medium, border-bottom-style:dashed, border-left-width:20px 등

☞ 속성값의 개수에 따른 적용 방향
4 → 상 우 하 좌
3 → 상 좌우 하
2 → 상하 좌우
1 → 상하좌우

 box-shadow 속성
박스 요소에 그림자 스타일 지정
none, 그림자

☞ 그림자
수평위치: 0보다 크면 오른쪽 0보다 작으면 왼쪽
수직위치: 0보다 크면 아래쪽 0보다 작으면 위쪽
흐림정도: 블러링되는 정도
퍼진거리: 음수 가능
색상: 기본은 검정색
inset: 박스 내부(안쪽) 그림자로 지정
