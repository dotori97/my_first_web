HTML요소: 미디어 및 시멘틱 요소

1. 미디어

(1) 오디오를 재생하는 audio요소

☞ 웹페이지에서 오디오를 재생하기 위한 요소
'<'audio src="파일이름" controls autoplay loop'>' '<'/audio'>'

☞ 속성
→ autoplay : 웹문서 로딩 시에 오디오를 자동으로 즉시 재생 여부
→ controls: 오디오의 제어버튼(재생, 일시정지, 불륨 등) 표시 여부
→ loop: 반복 재생 여부
→ muted: 음소거 여부
→ preload="값": 웹 페이지가 열릴 때 오디오의 로딩방식 지정
  • none: 재생 전까지 오디오 파일을 미리 로드하지 않음
  • auto: (기본) 웹 페이지 로딩 시 바로 전체파일을 자동으로 로드
  • metadta: 재생 전까지 오디오의 메타 파일(크기, 첫 프레임 등)만 로드
   ♡ autoplay 속성이 지정되면 preload 속성은 무시됨
→ src="URL": 재생활 오디오 파일의 URL

(2) 다양한 파일 형식을 지정하는 source 요소

☞ 하나의 미디어 소스에 대해 서로 다른 형식을 갖는 여러 개의 파일을 동시에 저장하기 위한 요소
→ audio/video 요소의 src 속성의 역할을 대신하는 것
→ 가장 위에서 나열된 source 요소의 파일부터 차례대로 재생 가능 여부를 검사해서 재생

☞ 속성
→ src="URL"
→ type="MIME 형식" : "audio/mp3", "audio/wav", "audio/ogg"
→ media="미디어 쿼리"

 예제

'<'audio controls autoplay loop'>'
'<'source src="song.ogg" type="audio/ogg"'>'
'<'source src="song.wav" type="audio/wav"'>'
'<'source src="song.mp3" type="audio/mp3"'>'
'<'/audio'>'

(3) 비디오를 재생하는 video 요소

'<'video src="비디오 파일 URL" 속성 ...'>'
'<'/video'>'

☞ 비디오형식: MP4(*.mp4), WebM(*.webm), Ogg(*.ogv)
☞ 속성
→ autoplay, controls, loop, muted, preload, src
→ width="픽셀", height="픽셀"
→ poster="이미지 파일의 URL" : 지정한 비디오 데이터가 설정되어 있지 않거나 비디오가 로딩되는 동안에 보여줄 이미지 지정

 속성

(4) 콘텐츠의 재생 구간 지정

☞ audio/video 또는 source 요소의 src 속성 이용해서 지정 src="파일명#t=[시작시간 시:분:초][,종료시간 시:분:초]"

 예시

시멘틱 요소

(1) 시멘틱 요소란?

☞ HTML문서에 의미 부여 가능. 검색엔진, 문서 해석기 등에 의한 자동적인 문서처리/해석도 가능
문서구조화 → header, nav, section, article, aside, footer
블록 요소 → main, figure~figcaption, details~summary
인라인 요소 → time, meter, progress, ruby~rt, wbr

(2) 문서 구조화를 위한 요소

☞ header → 머리말(로고, 사이트 이름 등)
☞ nav → 메뉴 영역
☞ section → 제목/주제별로 나눌 수 있는 문서의 내용 영역
☞ article → 개별 콘텐츠를 나타내는 요소
☞ aside → 좌우측의 보조 콘텐츠 영역
☞ footer → 꼬리말(제작자 정보 및 저작권 보호 등)

(2-1) 머리말과 꼬리말을 담는 header, footer 요소

 header요소
머리말. 사이트 이름, 로고와 소개 정보 등
본래의 역할은 머리말이지만 section등 내용 중간에서 사용되는 경우 머리글 역할
메타정보를 나타내는 데 사용되는 head요소와 헷갈리지 않게 유의

 footer요소
본래의 역할은 꼬리말이지만 내용 중간에 사용되는 경우 바닥글 역할

(2-2) 메뉴와 보조 콘텐츠를 담는 nav, aside 요소

 nav요소
☞ navigational links영역
주로 웹 페이지 전체에 적용되는 상단의 메뉴를 표시하기 위한 링크를 모아 놓은 곳

 aside요소
주요 콘텐츠 이외에 남아 있는 콘텐츠나 참고가 되는 콘텐츠 등을 나타낼 때 사용
주로 웹 페이지 왼쪽이나 오른쪽에 위치

 예시
        '<'body'>'

        '<'nav'>'

            '<'ol'>'

                '<'li'>''<'a href="..."'>'메일'<'/a'>''<'/li'>'

                '<'li'>''<'a href="..."'>'카페'<'/a'>''<'/li'>'

                 ..블로그, 쇼핑, 사전, 증권 등등

            '<'/ol'>'

        '<'/nav'>'

        '<'/body'>'

(2-3) 내용 영역 section, 개별 콘텐츠 article요소

seection 영역에 포함된 개별적인 콘텐츠가 article
하지만 article요소는 section요소와 분리하여 개별적으로 사용이 가능하며
articel 요소 안에 section 또는 article 요소 포함이 가능하다

(3) 시멘틱 블록 요소

☞ 주요 콘텐츠 블록을 지정하는 main 요소
☞ 삽입 그림을 블록화하는 figure 요소 → 하위 요소에는 figcaption
☞ 세부정보를 선택적으로 표시하는 details 요소 → 하위 요소에는 summary

(3-1) 주요 콘텐츠 블록을 지정하는 main요소

문서 body 영역의 주요 콘텐츠 블록을 지정할 때 사용
article, aside, header, footer, nav 요소의 하위 요소로 사용불가
브라우저 출력에는 아무런 영향을 미치지 않는다

(3-2) 삽입 그림을 블록화하는 figure요소

본문에서 삽입된 독립적인 콘텐츠(그림, 다이어그램, 사진, 소스코드, 동영상 등)를 하나로 묶어
블록을 형성하고, 캡션을 포함시켜 서로의 관계를 명확히 구조화시킬 때 사용

(3-3) 세부정보를 선택적으로 표시하는 details요소

사용자의 요구(핸들 클릭)에 따라 추가적인 세부정보를 보여주거나 숨기는 형태의 상호작용을 지원하는 요소
open 속성을 이용해 세부 정보가 사용자에게 보여지도록 지정할 수 있다

 summary요소
→ 세부 정보에 대한 캡션을 지정하는 details요소의 자식요소

(4) 시멘틱 인라인 요소

☞ 시간/날짜를 지정하는 time요소
☞ 주어진 값이나 비율을 나타내는 meter요소
☞ 작업의 진행률을 나타내는 progress요소
☞ 루비 주석을 지정하는 ruby요소 → 하위요소 rt
☞ 줄바꿈의 허용 위치를 지정하는 wbr요소

(4-1) 시간/날짜를 지정하는 time요소

시간/날짜 표현에 의미를 부여하기 위한 요소
datetime 속성 → 시스템이 인식할 수 있도록 정확한 형식의 시간/날짜를 지정

 예시

기상 시간은 '<'time'>'07:00'<'/time'>'이며 취침 시간은 '<'time'>'23:00'<'/time'>'입니다

HTML5 표준안은 '<'time datetime="2014-10-28"'>' 2014년 말'<'/time'>'에 발표되었다

↓

기상 시간은 07:00이며 취침 시간은 23:00입니다

HTML5 표준안은 2014년 말에 발표되었다

(4-2) 주어진 값이나 비율을 나타내는 meter요소

일정 범위 안의 측정값이나 분포 비율 등을 표시할 때 사용
측정기 gauge → 현재 디스크 사용량과 같이 일정 시점에서 주어진 정적인 양을 표시

☞ 속성
value → (필수) 실제록 측정한 현재의 값
min, max → 요소가 인식하는 최소값(0,0)과 최대값(1,0)
low, high → 허용되는 범위의 최소값/최대값 min ≤ low ≤ high ≤ max
optimum → 최적의 값

 예시

디스크 사용률'<'meter max=100 value=70'>''<'/meter'>' (1TB기준)

영화 예매율'<'meter min=0 max=100 value=95'>''<'/meter'>'

지방선거 투표율'<'meter value=0.35 title="오후 3시 현재 투표율"'>''<'/meter'>'

↓

디스크 사용률 (1TB기준)

영화 예매율

지방선거 투표율

(4-3) 작업의 진행률을 나타내는 progress요소

파일 복사나 다운로드 등과 같이 작업의 현재 진행 정도를 나타낼 때 사용
☞ 속성
value → 완료된 작업의 양을 나타내는 현재 상태의 값
max → 작업에 필요한 작업량의 최대값(1,0)
☞ 현재 진행 중인 상태 값의 지속적인 업데이트가 필요하므로 반드시 자바스크립트와의 연동이 필요

(4-4) 루비 주석을 지정하는 ruby 요소

루비주석 → 일본어/한자의 기본 문자열 주변에 발음법이나 의미를 나타내는 위해 추가되는 짧은 텍스트
ruby요소
하나 이상의 문자로 구성되는 기본 텍스트를 작성
rt요소
ruby요소의 자식요소

 예시
'<'ruby'>'HTML5 웹 프로그래밍 '<'rt'>'2학년1학기'<'/rt'>''<'/ruby'>'
↓
HTML5 웹 프로그래밍 2학년1학기

(4-5) 줄바꿈의 허용 위치를 지정하는 wbr요소

텍스트에서 줄바꿈을 허용할 위치를 지정할 떄 사용
긴 URL 또는 단어/텍스트를 표시할 떄 유용
