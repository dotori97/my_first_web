HTML요소: 웹 폼

1. form과 input요소

(1) 입력 요소를 담는 form 요소

사용자로부터 다양한 형태의 데이터를 입력받는데 필요한 요소들을 담는 컨테이너
form 내에서 사용해야 한다 ☞ input, textarea, select, datalist, output, fieldset, label 등

 속성
action="URL" → 폼 데이터를 전달받아 처리할 서버 응용프로그램의 주소
method="전송방식" → 폼 데이터를 서버로 전송하는 방식
☞ 속성값에는 get과 post가 있다
name(폼 이름), accept-charset(문자 인코딩 방식), autocomplete(자동완성 기능의 사용 여부),
enctype(method="post"일 때 데이터 인코딩 방식), novalidate(데이터의 유효성 검사하지 않음),
target(서버로부터 전달받은 처리 결과를 보여줄 창)

(2) get vs post

 method="get"
폼데이터가 이름과 값의 쌍 형태로 URL에 포함되어 전송
보안이 중요하지 않은 데이터(예: 검색어)
'이전 페이지' 버튼을 통한 이동이 가능

 method="post"
HTTP Request 몸체 속에 포함시켜 전송
보안이 필요한 정보 등의 전송에 사용

(3) input

input type="입력타입" 속성="값" ... → <> 생략



type 속성 값으로 다양한 유형의 입력 타입이 있다

&#e61e; 속성
text, password, radio, checkbox, hidden(숨겨진 상태로 입력), file, image,
submit(제출 버튼), reset, button, search, tel, url, email, number(특정 범위의 숫자),
range(특정 범위의 숫자), color, date, month, week, time, datetime-local(날짜와 시간)

2. 데이터의 입력과 선택

(1) 텍스트 입력

 text
input type="text"... → <> 생략

한 줄에 해당하는 짧은 텍스트를 입력할 떄 사용

 textarea
여러 줄에 해당하는 텍스트를 입출력할 떄 사용
속성 → rows, cols, name, wrap, readonly

 기본 사용 형식
textarea rows="숫자" cols="숫자"... → <>생략
텍스트 영역에 표시할 내용
/textarea → <>생략

 속성
name="이름" → 텍스트 영역의 이름
rows="숫자" → 텍스트 영역의 높이(라인 수)
cols="숫자" → 텍스트 영역의 너비(한 즐에 입력할 수 있는 글자 수)

autofocus → 입력창이 진하고 입력 위치에서 커서 껌뻑인다
value → 텍스트에서 value는 초기에 표시되는 텍스트
placeholder → 문자열을 보여줌으로서 사용자가 잘못 입력하는 것을 방지하기 위해 사용
disabled → 비활성화

(2) 특정 형식의 텍스트 입력

비밀번호 입력 → input type="password"... (<>생략)
검색어 입력 → input type="search"... (<>생략)
전화번호 입력 → input type="tel" ... (<>생략)
url 입력 → input type="url" ...(<>생략) → 자동 유효성 검사
이메일 주소 입력 → input type="email" ...(<>생략) → 자동 유효성 검사

 예시
form id="frm_id" → <>생략 이름: input type="text"
비밀번호: input type="password"
검색어: input type="search" value="HTML"
전화번호: input type="tel" required(누락 시 요청)
URL: input type="url" placeholder="http://www.knou.ac.kr"
이메일 주소: input type="email" multiple(여러 개 입력 가능)
input type="submit" form="frm_id" → form요소 안에서 사용해야 하지만 id를 이용하면 가능

(4) 항목 선택

라디오 버튼 → 오직 1개 항목만 선택
input type="radio"

체크박스 → 여러 항목의 동시 선택 가능
input type="checkbox"

☞ 주요 속성
name="이름" → 같은 그룹에 속하는 항목은 동일한 속성값 부여
value="텍스트" → 서버로 전달되는 값(name="value")으로 각 항목마다 서로 다르게 부여되는 항목 내용
checked → 초기값으로 선택되는 항목 지정

파일 선택

자신의 시스템에 있는 파일을 선택하는 버튼 생성
☞ input type="file"...

 속성
accept="파일유형" → 선택할 수 있는 파일의 유형 지정
multiple → 동시에 여러 파일을 선택할 수 있도록 지정

 예시
form enctype="multipart/form-data"
   input type="file" accept="image/*" multiple
/form



데이터 리스트에서의 선택

주어진 데이터 옵션 리스트로부터 원하는 데이터 선택

☞ select 요소
→ 드롭다운 리스트를 만드는 경우

☞ datalist 요소
→ input type="text"..에 대한 옵현 리스트 지정
→지정한 텍스트 타입이 포커스를 받으면 미리 입력한 옵션이 드롭다운 형태로 표시

☞ option 요소
→ select/datalist 요소의 하위 요소로 옵션 리스트의 각 옵션 정의

드롭다운 리스트를 만드는 select요소

๡ 사용형식
select 속성="값"...
   option value="값"... 항목1 /option
   option value="값"... 항목2 /option
...    option value="값"... 항목n /option
/select

 속성
name="이름" → 드롭다운 리스트의 이름
size="숫자" → 한 번에 화면에 보이는 항목의 개수
multiple → 여러 항목을 한 번에 선택할 수 있도록 지정
autofocus, disabled, form, required

옵션 리스트의 각 옵션을 정의하는 option요소

 속성
disabled → 해당 옵션의 비활성화
selected → 페이지가 로드될 떄 미리 선택될 초기 옵션 지정
value="텍스트" → 서버로 전달되는 항목의 값

 예시

form
 과목
 select name="subject" size="1"
    option value="Java"> Java 프로그래밍 /option
    option value="html" selected> HTML5 웹 프로그래밍 /option
    option value="algo"> 알고리즘 /option
    option value="dc"> 디지털 논리회로 /option
    option value="db"> 데이터베이스 시스템 /option
  /select /form
