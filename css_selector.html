CSS: 개요 선택자 색상

1. CSS 개요

(1) CSS란?

Cascading Style Sheets
웹 문서에 스타일(글꼴, 색상, 여백 등)을 추가하는 간단한 메커니즘

(2) CSS 형식

선택자 { 선언 블록 }

선언블록 → 선언; 선언; ...
선언 → 속성:값;
☞ 예시
p{color:red; font-size:25px;}

(3) CSS 적용 방법

HTML 문서에서 스타일이 정의되는 위치에 따른 구분
☞ style 요소 사용 → 임베디드 방식
head
  style
   p{color:red; forn-size:25px;}
  /style
/head

☞ style 속성 사용 → 인라인 방식
p style="color:red; font-size:25px;" CSS /p

☞ 외부 파일 사용 → 링크 방식
ex.css → p{color:red; font-size:25px;}
head
  link rel="stylesheet" href="ex.css"
/head

style
 @import url(ex.css)
/style

(4) CSS 적용 우선순위

style속성 > style요소 > 외부파일 > 브라우저 디폴트

2. 선택자

(1) 선택자란?

스타일을 적용하려는 HTML요소를 지정하기 위한 패턴
기본선택자, 가상선택자(가상클래스, 가상요소), 결합자

 기본선택자의 종류
전체 선택자 → 모든 요소를 선택
타입 선택자 → 지정한 모든 요소만 선택
클래스 선택자 → class="속성명" 속성을 가진 요소만 선택
아이디 선택자 → id="속성명" 속성을 가진 요소만 선택
속성 선택자 → 속성 표현에 해당하는 속성을 가진 요소 선택

기본 선택자

 전체 선택자
'*' 기호 사용 → 웹 문서의 모든 요소에 스타일 적용
*{color:green}

 타입 선택자
스타일을 적용하려는 HTML 요소 이름을 직접 사용
table{color:blue}
ul{color:red; font-style:italic;}

 클래스 선택자
☞ 같은 요소지만 서로 다른 스타일을 적용하는 경우
css 정의 → 요소명.클래스명{속성:값;...}
css 적용 → <요소명 class="클래스명"> 내용
☞ 서로 다른 요소지만 같은 스타일을 적용하는 경우
css 정의 → .클래스명{속성:값; ...} → 요소명 생략
css 적용 → <요소명 class="클래스명"> 내용

 아이디 선택자
클래스 선택자의 사용방법과 거의 동일 → #기호, id속성
css 정의 → 요소명#아이디명{속성:값; ...}
css 적용 → <요소명 id="아이디명"> 내용
☞ 아이디명은 문서 내부에서 중복되지 않고 오직 하나의 요소에서만 사용

☞ 예시
div요소는 아이디 선택자(style1) 적용

h3요소에는 class2와 header+style1

첫번째 p요소에는 p, class1, class2+style1

두번째 p요소에는 p, style2, class2+style1 적용


 속성 선택자
요소의 속성값에 대한 표현과 일치하는 HTML요소 선택
☞ 요소명[속성명 연산자 값]{속성:값; ...}

요소명[속성명] → 속성값과 상관없이 해당 속성을 사용하는 요소 선택
요소명[속성명="값"] → 속성값과 일치하는 요소 선택
요소명[속성명~="값"] → 속성값을 공백으로 구분한 요소 선택
요소명[속성명|="값"] → 속성값이 정확히 일치하거나 값으로 시작하고 바로 뒤에 - 기호로 구분한 요소 선택
요소명[속성명^="값"] → 속성값의 시작 부분의 문자와 일치하는 요소 선택
요소명[속성명$="값"] → 속성값의 끝부분의 문자와 일치하는 요소 선택
요소명[속성명*="값"] → 속성갑 전체 중 어떤 일부분이라도 일치하는 요소 선택

가상선택자

웹 문서상에는 존재하지 않지만 어떤 상황이 발생한 경우에만 적용

☞ 가상 클래스 → 기본선택자:가상클래스
링크 → :link, :visited,
사용자 동작 → :hover, :active, :focus
UI상태 → :enabled, :disabled, :checked
부정 → :not
트리구조 → :root, :nth-child(), :nth-last-child(), :first-child, :last-child, :nth-of-type(), :first-of-type; :only-child, :only-of-type, :empty

:link 아직 방문하지 않은 모든 링크 선택
:visited 방문한 적이 있는 링크 선택
:hover 마우스가 올라간 링크 선택
:active 활성화되는(클릭되는) 링크 선택

:focus 포커스를 가진 요소
:enabled 활성화된 상태의 요소
:disabled 비활성화된 상태의 요소
:checked 체크된 input 요소(체크박스, 라디오버튼)
:not(x) x로 표현되지 않은 요소
:nth-child(n) 동일한 부모 요소 안에서 n번째 자식 요소 선택
:nth-last-child(n) 동일한 부모 요소 안에서 끝에서 n번째 자식 요소 선택
:nth-of-type(n) 동일한 부모 요소 안의 형제 요소들 중에서 지정한 요소만을 계산하여 n번째 요소 선택
:first-of-type = :nth-of-type(1)

☞ 가상 요소 → 기본선택자::가상요소
::first-letter, ::first-line, ::before, ::after

Love is patient, ...

가상 요소 ...


::first-line 특정 요소의 첫 번째 줄을 선택
::first-letter 특정 요소의 첫 번째 글자를 선택
::before 요소 앞에 내용/스타일 추가
::after 요소 뒤에 내용/스타일 추가

결합자

두 개 이상의 선택자를 결합하는 방법
h1, p
결합자 이름사용 형식예제
자손(하위)결합자선택자1 선택자2div li
자식 결합자선택자1 > 선택자2p > em
인접 형제 결합자선택자1 + 선택자2h1 + p
일반 형제 결합자선택자1~선택자2li.style1~li
그룹 결합자선택자1, 선택자2, ...

CSS 결합자

css에서 사용하는 결합자의 종류를 연습하기 위한 문서입니다

  • 자손 결합자는 선택자1 공백 선택자2
  • 자식 결합자는 선택자1 > 선택자2
  • 인접 형제 결합자: 선택자1 + 선택자2
  • 일반 형제 결합자: 선택자1 ~ 선택자2
  • 그룹 결합자: 선택자1, 선택자2, ...

3. 색상

색상에 해당하는 영어 단어를 직접 사용
16진수 사용: #RRGGBB → RR, GG, BB는 00~ff(0~255) 사이의 16진수 값
RGB함수 사용:
rgb(R, G, B) → R, G, B 각 값을 0~255 사이의 10진수로 표시
rgb(R%, G%, B%) → R, G, B 각 값을 백분율(0~100%)료 표시
rgba(R, G, B, Alpha): rgb(R, G, B) + 투명도 → 완전 투명(0.0)~완전 불투명(1.0)
HSL함수 사용:
hsl(색조, 채도, 밝기) → 색조(red=0=360, green=120, blue=240) 채도(0~100%) 밝기(0~100%)
hsla(색조, 채도, 밝기, Alpha)

 색상 지정 관련 속성
color: HTML요소의 텍스트 색상 (값 → <색상>, transparent=rgba(0,0,0,0.0))
background-color: HTML요소의 배경색
border-color: 테두리 색상
opacity: 요소의 불투명도 (값 → 0.0~1.0)
