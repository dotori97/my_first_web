HTML요소: 웹 폼

1. form과 input요소

(1) 입력 요소를 담는 form 요소

사용자로부터 다양한 형태의 데이터를 입력받는데 필요한 요소들을 담는 컨테이너
form 내에서 사용해야 한다 ☞ input, textarea, select, datalist, output, fieldset, label 등

 속성
action="URL" → 폼 데이터를 전달받아 처리할 서버 응용프로그램의 주소
method="전송방식" → 폼 데이터를 서버로 전송하는 방식
☞ 속성값에는 get과 post가 있다
name(폼 이름), accept-charset(문자 인코딩 방식), autocomplete(자동완성 기능의 사용 여부),
enctype(method="post"일 때 데이터 인코딩 방식), novalidate(데이터의 유효성 검사하지 않음),
target(서버로부터 전달받은 처리 결과를 보여줄 창)

(2) get vs post

 method="get"
폼데이터가 이름과 값의 쌍 형태로 URL에 포함되어 전송
보안이 중요하지 않은 데이터(예: 검색어)
'이전 페이지' 버튼을 통한 이동이 가능

 method="post"
HTTP Request 몸체 속에 포함시켜 전송
보안이 필요한 정보 등의 전송에 사용

(3) input

input type="입력타입" 속성="값" ... → <> 생략



type 속성 값으로 다양한 유형의 입력 타입이 있다

&#e61e; 속성
text, password, radio, checkbox, hidden(숨겨진 상태로 입력), file, image,
submit(제출 버튼), reset, button, search, tel, url, email, number(특정 범위의 숫자),
range(특정 범위의 숫자), color, date, month, week, time, datetime-local(날짜와 시간)

2. 데이터의 입력과 선택

(1) 텍스트 입력

 text
input type="text"... → <> 생략

한 줄에 해당하는 짧은 텍스트를 입력할 떄 사용

 textarea
여러 줄에 해당하는 텍스트를 입출력할 떄 사용
속성 → rows, cols, name, wrap, readonly

 기본 사용 형식
textarea rows="숫자" cols="숫자"... → <>생략
텍스트 영역에 표시할 내용
/textarea → <>생략

 속성
name="이름" → 텍스트 영역의 이름
rows="숫자" → 텍스트 영역의 높이(라인 수)
cols="숫자" → 텍스트 영역의 너비(한 즐에 입력할 수 있는 글자 수)

autofocus → 입력창이 진하고 입력 위치에서 커서 껌뻑인다
value → 텍스트에서 value는 초기에 표시되는 텍스트
placeholder → 문자열을 보여줌으로서 사용자가 잘못 입력하는 것을 방지하기 위해 사용
disabled → 비활성화

(2) 특정 형식의 텍스트 입력

비밀번호 입력 → input type="password"... (<>생략)
검색어 입력 → input type="search"... (<>생략)
전화번호 입력 → input type="tel" ... (<>생략)
url 입력 → input type="url" ...(<>생략) → 자동 유효성 검사
이메일 주소 입력 → input type="email" ...(<>생략) → 자동 유효성 검사

(8) 버튼 생성

input요소의 type 속성은 4가지 형태의 버튼 제공
제출버튼(submit), 리셋버튼(reset), 이미지버튼(image), 일반버튼(button)
value="버튼명" → 버튼 위에 표시되는 이름

☞제출버튼 → input type="submit"
입력한 데이터를 form요소의 action속성에서 지정한 서버로 전송
기본 버튼명(value 속성 미사용) → "제출"(크롬의 경우)

☞ 리셋버튼 → input type="reset"
입력하고 있는 데이터를 모두 지우고 초기화
기본 버튼명 → "초기화" (크롬의 경우)

☞ 이미지버튼 → input type="image"
이미지를 사용해서 제출 버튼의 기능을 만들 때 사용
속성 → src, width, height, alt

☞ 일반버튼 → input type="button"
클릭을 통해 자바스크립트 코드를 수행하는 경우에 사용

 button 요소
button type="button/submit/reset" name="이름" value="초기값"
  버튼 위에 표시될 텍스트나 이미지를 지정
/button

3. 기타 요소

폼 내의 값을 출력하는 output요소

폼 내의 다양한 값을 출력할 수 있도록 지정하는 요소 특정 input요소에 입력된 값이 변하게 되면 그것을 곧바로 반영하여 화면에 출력할 때 사용

input요소의 값이 변하면
→ form요소의 input 이벤트 발생
→ 이벤트에 대한 처리로서 자바스크립트 문법으로 출력할 값의 연산식을 지정

A * B =

폼 요소와 캡션을 묶어주는 label요소

input요소에 대한 레이블을 정의
시각적 효과는 없음. 단지 마우스 사용자의 사용성을 향상시킴
label요소 내에 있는 텍스트(캡션)를 클릭

폼 요소를 그룹핑하는 fieldset요소

폼에서 사용되는 관련 요소들을 그룹핑할 때 사용
시각적으로 관련 요소들을 포함하는 사각형 테두리가 표시됨
속성 → name, form, disabled

☞ legend요소
fieldset요소의 하위 요소
그룹핑되는 관련 요소들에 대한 캡션을 표시할 때 사용
사각형 테두리 선상에 캡션이 표시됨
