HTML요소: 텍스트 표현

1. 줄을 바꾸는 br요소

편집기에서 엔터키 같은 기능
종료태그 없이 사용

2. 단락을 나누는 p요소

문단을 나누어 독립된 단락을 생성
단락 구분을 위해 단락 전후에 빈 줄이 자동으로 추가됨

3. 수평선을 그리는 hr요소

텍스트 사이에 수평선을 그림. 주제가 변경되는 콘텐츠를 구분하는 용도
자동 줄바꿈, 종료태그 없이 사용

4. 단락의 제목을 지정하는 hn요소

굵은 글자(볼드체), 자동 줄바꿈

5. 텍스트를 자유롭게 표시하는 pre요소

편집기에서 문자가 입력된 형태를 그대로 유지하여 출력
텍스트는 고정폼. 행의 길이가 브라우저 길이보다 길어져도 자동 줄바꿈되지 않음
특수 문자가 많은 텍스트를 표시할 때 유용
pre요소 안에서 html요소는 문자로 취급되지 않고 html요소로서 적용됨

6. 인용문을 작성하는 blockquote요소

상하 빈 줄로 구분되는 인용문 단락의 생성 및 들여쓰기
cite속성으로 인용된 부분의 URL지정

7. 공백 및 특수문자 입력

두 칸 이상의 연속된 공백, 예약어로 사용되는 문자, 키보드로 직접 입력할 수 없는 문자를 입력하는 방법
&특수이름; 또는 10진수코드(앞에 &#) 또는 16진수코드(앞에 &#x)
예:
줄바꿈 요소→ <br>
  문단 구분 요소 → <p>
    공백문자 → & nbsp;
      키보드로 입력 불가 문자 → ©,♡,☞ 등

8. 글자 스타일 지정 요소

 ☞ 물리적 스타일 관련 요소

→ 브라우저에 표시되는 출력 모양 지정
→ b(강조), i(기울기), s(가운데줄), u(아래줄), sup(위첨자), sub(아래첨자), small(작게) 등

 ☞ 논리적 스타일 관련 요소

→ 요소 자체가 의미를 갖는다
→ cite, code, samp, var, dfn, em(강조), strong(중요성), abbr(축약), address, kbd(키보드), "q"(인용), mark 등

9. 블록 요소 vs 인라인 요소

☞ 블록 요소

→ 문서구조 요소
→ 항상 새로운 줄에서 시작해서 하나의 줄 전부 차지해서 내용 표시되는
→ 자동으로 줄바꿈 수행
→ div, p, h1 등

☞ 인라인 요소

→ 텍스트 요소
→ 입력하는 내용만큼의 공간 차지해서 내용 표시
→ 줄바꿈 미수행
→ span, img, strong 등

10. 콘텐츠 컨테이너 요소: div, span

☞ div 요소

→ 블록 요소
→ 대량의 내용에 대해 css 스타일 지정 또는 전체 페이지 공간을 분할하여 레이아웃을 구성하는데 사용

☞ span 요소

→ 인라인 요소
→ 텍스트 일부에 대해 css 스타일을 지정 또는 자바스크립트에서 조작할 때 유용
