1. 리스트

(1) Unordered List

☞ 순서 없는 리스트를 지정
☞ ul ~ li

(2) Ordered List

☞ 순서 있는 리스트를 지정
☞ ol ~ li
☞ 속성
→ 순서가 부여된 기호의 유형 지정 (type="값")
→ 항목의 시작 번호 지정 (start="숫자")
→ 항목 번호를 역순으로 지정 (reversed)

(3) Description List

☞ 서술 리스트를 지정
☞ dl ~ dt/dd
☞ dt요소 → 용어나 이름을 나열. 자동 줄바꿈
☞ dd요소 → 용어나 이름에 대한 설명을 제공할 때 사용. 자동 줄바꿈, 들여쓰기

(4) ul, ol 내에서 사용되는 자식요소 li

☞ 들여쓰기, 자동 줄바꿈, 종료태그 없이 사용가능
☞ 속성 → 항목 번호를 중간에 바꿀 때 사용 (value="숫자")

2. 이미지

이미지를 삽입하는 img요소♡

☞ 이미지는 인라인 요소
☞ 속성
→ src="파일의 URL" (이미지 파일 경로)
→ width="숫자" (픽셀, %)
→ height="숫자" (픽셀, %)
→ alt="텍스트" (이미지가 표시되지 않을 때 대체 표시될 텍스트)
→ usemap="#맵이름" (클라이언트 측의 이미지맵)

3. 하이퍼링크

(1) 하이퍼링크를 삽입하는 a 요소

 기본 형식
'<'a href="이동할 위치"'>' 링크를 걸어줄 내용물(텍스트,이미지) '<'/a'>'
 속성
→ href="URL"
→ target="창이름"(_blank, _self)
→ download

(2) id 속성을 활용한 특정 위치로의 이동

동일한 페이지 또는 다른 페이지에서 특정 위치로 이동하는 경우

링크부분
☞ 같은 페이지 내에서 이동하는 경우 → '<'a href=#이동할 특정 위치의 이름"'>' 내용 '<'/a'>'
☞ 다른 페이지의 특정 위치로 이동하는 경우 → '<'a href="페이지경로#이동할 특정 위치의 이름"'>' 내용 '<'/a'>'

이동할 특정 위치
☞ '<'a id ="이동할 특정 위치의 이름"'>' 내용 '<'/a'>'

(3) 이미지맵

☞ 하나의 이미지를 여러 개의 영역으로 나누어 각 영역마다 링크를 지정하는 것
'<'img src="URL" usemap="#맵이름"'>'

'<'map name="맵이름"'>'
'<'area
    shape="이미지맵에서 각 영역의 모양(default, rect, circle, poly)"
    coords="각 영역을 구성하는 좌표값"
    href="URL"
    target="링크된 내용이 열릴 창"
    download="href로 지정된 파일을 다운로드"
    alt="대체 텍스트 내용"'>'
'<'/map'>'

 예시

(4) 새로운 창을 삽입하는 iframe 요소

 현재 웹 페이지 안에서 다른 웹 페이즈를 표시하는 창을 삽입할 때 사용
 속성
→ name="이름" (창의 이름. target 속성 값으로 사용)
→ src="URL"
→ srcdoc="HTML코드"
→ width="폭"
→ height="높이"

 예시

3. 테이블

(1) 기본 형식

๡ 예시
☞ 전체적으로 table로 감싸고 caption 다음부터 각 열을 tr로 감쌈

과목별 성적(caption으로 감쌈)
과목(th)중간시험(th)기말시험(th)
머신러닝(td) 30(td) 46(td)
알고리즘(td) 29(td) 50(td)

(2) 테이블 생성을 위한 기본 요소

 table요소
m행xn열 → m개의 tr요소, 각 tr내에서 n개의 td요소
테이블의 테두리 지정 → border속성, 테두리 스타일 지정은 css 사용

 tr요소(table row)
테이블에서 행을 정의하는 요소로 td요소 또는 th요소 포함해야 사용 가능

 td요소(table data)
데이터(텍스트, 이미지, 테이블 등)를 표시하는 표준 셀을 정의
왼쪽 정렬

 th요소(table header)
테이블 내부에서 행열의 제목을 표시하는 헤더 셀을 정의
진하게 가운데 정렬

 caption요소
테이블 전체에 대한 제목을 표시할 때 사용
테이블 상단에 진하게 가운데 정렬. css속성으로 변경 가능

(3) 셀 병합

☞ 인접한 여러 셀을 합쳐서 하나의 셀로 만드는 경우
→ td 또는 th 요소 내에서만 사용

 rowspan속성
→ rowspan="n"
→ 현재 셀 위치에서 인접한 n개의 행을 병합(세로 방향)

 colspan속성
→ colspan="n"
→ 현제 셀의 위치에서 인접한 n개의 열을 병합(가로 방향)

 예시

☞ 병합 전

1-11-21-31-4
2-12-22-32-4
3-13-23-33-4

☞ 병합 후

1-1 ~ 1-4
2-1
3-1		 2-2
3-2		 2-3 2-4
3-3, 3-4

 colgroup요소
☞ 하나 이상의 열들을 모아서 한꺼번에 스타일 지정
